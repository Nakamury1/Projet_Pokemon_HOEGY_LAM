Réalisation d'une simulation de combats Pokemon
- Possibilité de choisir le pokemon que l'on souhaite
- Possibilité d'appuyer sur le bouton attaque ou capacité spéciale pour attaquer le pokemeon adverse
- Possibilité de soigner notre pokemon

Outils utilisés: PHP, SQL

Projet réalisé en groupe dans le cadre de l'école
